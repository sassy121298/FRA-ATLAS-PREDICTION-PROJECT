## Overview

FRA Atlas is a machine learning and GIS-based project designed to assess forest-related risks using environmental and spatial data. The system combines predictive modeling, decision support, and interactive web-based GIS visualization to support data-driven forest management and risk mitigation.

---

## Key Features

* Forest risk prediction using machine learning and deep learning models
* Data preprocessing, feature engineering, and exploratory data analysis (EDA)
* Model evaluation using accuracy and precision metrics
* Decision Support System (DSS) for generating forest risk recommendations
* Interactive Web GIS visualization of forest risk zones

---

## Technologies & Tools

* **Programming Language:** Python
* **Data Processing:** Pandas, NumPy
* **Machine Learning:** Random Forest Classifier
* **Deep Learning:** Convolutional Neural Networks (CNN) using TensorFlow & Keras
* **Model Evaluation:** Accuracy Score, Precision Score, Stratified K-Fold Cross-Validation
* **Hyperparameter Tuning:** GridSearchCV, RandomizedSearchCV
* **GIS & Visualization:** Folium, MarkerCluster
* **Other Tools:** OpenCV, Google Colab, GitHub

---

## Project Workflow

1. Data collection and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature encoding and scaling
4. Machine learning and deep learning model training
5. Model evaluation and validation
6. Risk-based decision support generation
7. Interactive Web GIS map creation

## Decision Support System (DSS)

The Decision Support System analyzes model predictions and generates actionable recommendations related to forest risk levels. This helps support informed decision-making for forest monitoring, planning, and risk mitigation.

---

## Web GIS Visualization

The project includes an interactive web-based GIS map developed using Folium. The map visualizes forest risk zones with markers and clustering, enabling intuitive spatial analysis of high-risk and low-risk areas.
